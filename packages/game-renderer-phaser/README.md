# packages/game-renderer-phaser

Domain-neutral Phaser renderer for tile-based multiplayer worlds.

This package owns the Phaser host, scene lifecycle, tile layers, object sprites,
entity/avatar rendering, zone presentation, interaction markers, camera control,
DOM world overlays, renderer telemetry, and visual QA surfaces. Product apps
provide maps, asset catalogs, entity lists, interaction labels, asset-pack
configuration, and UI callbacks.
