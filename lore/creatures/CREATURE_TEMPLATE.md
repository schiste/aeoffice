### Entry Type Classification

**SPECIES** — A full creature category encompassing all individuals of that type.
- Example: *Les Lapins Rouges* — All mutated rabbits across Europe
- Use when: Documenting a new creature type for the first time
- Population: Typically thousands to millions

**BREED** — A regional variant, subspecies, or specialized adaptation within a species.
- Example: *Alpine Lapin Rouge* — Mountain-adapted variant with thicker fur
- Use when: A distinct population has diverged enough to warrant separate documentation
- Population: Typically hundreds to thousands
- **Required field:** `Parent Species: [link to species entry]`

**UNIQUE** — A specific named individual creature, often legendary or plot-relevant.
- Example: *Dweller Bane* — That one specific rat in that one specific cave
- Use when: A single creature has earned a reputation, name, or narrative significance
- Population: Exactly one (that's what "unique" means)
- **Required fields:**
  - `Parent Species: [link to species entry]`
  - `Parent Breed: [link if applicable, or "Standard"]`
  - `Current Location: [where it lives/was last seen]`
  - `Status: [Alive / Dead / Unknown / "Probably Fine"]`

### Danger Rating Scale Reference

#### Aggression (Unprovoked) — *"Will it start trouble?"*

| Rating | Description | Behavior |
|--------|-------------|----------|
| **0** | Pacifist | Actively avoids all contact; apologizes with body language while fleeing |
| **1** | Timid | Flees from humans unless cornered, starving, or having a really bad day |
| **2** | Wary | Maintains distance; will flee if approached; judges you silently |
| **3** | Territorial | "This is MY rotting barn and you can't have it" energy |
| **4** | Opportunistic | Will attack vulnerable humans unprovoked; knows what "vulnerable" looks like |
| **5** | Predatory | Actively hunts humans; has preferences about which ones |

#### Aggression (Provoked) — *"How does it respond to threats?"*

| Rating | Description | Behavior |
|--------|-------------|----------|
| **0** | Doormat | Does not fight back even when attacked; has given up on life |
| **1** | Sensible | Always chooses escape; has correctly assessed the post-Silence survival meta |
| **2** | Defensive | Brief retaliation then disengages; believes in proportional response |
| **3** | Petty | Fights until threat retreats; will make a point about it |
| **4** | Vindictive | Continues attacking after threat neutralized; "and STAY down" |
| **5** | Unhinged | Fights to the death; pursues fleeing attackers; writes your name on a list |

#### Physical Damage Potential — *"How badly can it hurt you?"*

| Rating | Description | Typical Injuries |
|--------|-------------|------------------|
| **0** | Adorable | Cannot injure adult human; may injure your feelings by not liking you |
| **1** | Annoying | Scratches, small bites; "I'll walk it off" injuries |
| **2** | Inconvenient | Deep cuts, punctures; "actually maybe we should bandage this" |
| **3** | Serious | Broken bones, severe lacerations; "I can see things I shouldn't" |
| **4** | Catastrophic | Dismemberment possible; "that used to be attached to me" |
| **5** | Terminal | Can kill healthy adult in single strike; closed casket situation |

#### Venom/Poison — *"Is there something extra in those wounds?"*

| Rating | Description | Effects |
|--------|-------------|---------|
| **0** | Clean | No toxic capability; just good honest violence |
| **1** | Spicy | Localized pain, mild swelling; like a very angry bee |
| **2** | Memorable | Intense pain, numbness; you'll be telling this story for years |
| **3** | Educational | Temporary paralysis, fever; teaches you about your own mortality |
| **4** | Transformative | Tissue necrosis, systemic effects; changes your relationship with that limb |
| **5** | Philosophical | Rapid lethality; plenty of time to contemplate your choices |

#### Pack Coordination — *"Do they work together?"*

| Rating | Description | Behavior |
|--------|-------------|----------|
| **0** | Antisocial | Never groups; actively dislikes own species; relatable |
| **1** | Reluctant | Groups for mating only; immediately regrets it |
| **2** | Coworkers | Groups share space; flee individually; no loyalty |
| **3** | Family | Basic group tactics; sentinels; will avenge each other (eventually) |
| **4** | Organized | Flanking, distractions, waves; has clearly been practicing |
| **5** | Terrifying | Complex strategy; adapts to your tactics; holds post-battle debriefs |

#### Stealth/Ambush Capability — *"Will you see it coming?"*

| Rating | Description | Detection Difficulty |
|--------|-------------|---------------------|
| **0** | Theatrical | Large, loud, visible; announces itself; may have theme music |
| **1** | Casual | Makes noise; visible in open; not trying very hard |
| **2** | Competent | Can approach unnoticed if you're distracted by literally anything |
| **3** | Professional | Skilled at concealment; you'll see it, but only after it's too late |
| **4** | Exceptional | Perfect camouflage or infinite patience; already behind you |
| **5** | Conceptual | You won't know it's there; survivors describe "a feeling of wrongness" |

#### Speed/Pursuit — *"Can you outrun it?"*

| Rating | Description | Comparison |
|--------|-------------|------------|
| **0** | Decorative | Cannot pursue; more of a location than a creature |
| **1** | Leisurely | Slower than walking human; you could moonwalk away from this |
| **2** | Moderate | Matches walking human; jogging escapes; it looks disappointed |
| **3** | Athletic | Matches running human; hope you stretched this morning |
| **4** | Unfair | Faster than sprinting human; cardio won't save you |
| **5** | Absurd | Catches anything; witnesses describe "a blur, then screaming" |

#### Intelligence — *"How smart is it?"*

| Rating | Description | Capabilities |
|--------|-------------|--------------|
| **0** | Philosophical | Pure reflex; no learning; has achieved freedom from thought |
| **1** | Learning | Simple conditioning; knows "this hurts, don't do this" |
| **2** | Aware | Learns from experience; recognizes you specifically; concerning |
| **3** | Clever | Problem-solving; uses environment; opens doors (oh no) |
| **4** | Scheming | Plans ahead; sets traps; remembers your face; tells friends |
| **5** | Smug | Near-human reasoning; communication; knows it's smarter than you |
