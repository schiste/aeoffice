## Overview

This folder contains detailed reference files for all character attributes used in post-Silence character creation. These resources help AI agents and content writers generate consistent, rich characters for the era 300 years after The Silence destroyed modern civilization.

Each trait file includes:
- Detailed options with dialog markers
- Example phrases and speech patterns
- Relationship to sound, survival, and community
- Cross-references to related traits

## Usage Guide

### For AI Agents

When generating a post-Silence character:

1. **Establish Origin:** Lineage and settlement type create the foundation
2. **Set Generation:** How far from "Before" shapes worldview
3. **Define Role:** Primary role determines daily life and vocabulary
4. **Add Mastery:** Skill level affects confidence and authority
5. **Layer Beliefs:** Views of Crystals, Silence, and Before shape philosophy
6. **Ground in Experience:** Silence exposure is often defining
7. **Position Socially:** Community standing determines influence

### For Content Writers

When writing dialog:

1. **Check the dialect file** for speech patterns
2. **Note role-specific vocabulary** from primary_role.md
3. **Consider generation effects** on knowledge and reference
4. **Verify worldview consistency** across belief files
5. **Use the Writing Checklist** in the main template

## Key World Details

### The Crystals
- Generate protective resonance
- Maintained by Crystal Tenders
- Survival depends on them
- Views range from sacred to scientific

### The Silence
- Lethal phenomenon in areas without sound
- Kills almost everyone exposed
- Resilients can survive exposure
- Source of all isolation

### Settlements
- Bubbles of protection
- Varied types (cave, coastal, crystal, frontier)
- Sound maintenance is constant
- Community survival requires cooperation

### Before
- The world that ended in 2034
- Mythologized, mourned, or dismissed
- Artifacts and stories remain
- Knowledge degrades across generations
