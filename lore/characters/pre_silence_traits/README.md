## Overview

This folder contains detailed reference files for all character attributes used in pre-Silence character creation. These resources help AI agents and content writers generate consistent, rich characters for the Sleepless in Decibels era.

Each trait file includes:
- Detailed options with dialog markers
- Example phrases and speech patterns
- Relationship to conspiracy involvement
- Cross-references to related traits

## Usage Guide

### For AI Agents

When generating a pre-Silence character:

1. **Start with Background:** Establish social class, upbringing, and cultural heritage first
2. **Add Education & Career:** These shape vocabulary and problem-solving approach
3. **Define Language:** Determine native language and communication style
4. **Layer Personal Traits:** Add temperament, intelligence type, and humor
5. **Ground in Parenting:** Sleep deprivation and noise complaints are central
6. **Position in Conspiracy:** Determine their role and knowledge level

### For Content Writers

When writing dialog:

1. **Check the trait file** for specific vocabulary and speech patterns
2. **Note the "Dialog Markers"** section for example phrases
3. **Consider trait interactions** (exhausted + choleric = explosive)
4. **Verify cross-references** for consistency
5. **Use the Writing Checklist** in the main template

## Cross-Reference System

Each trait file includes a "Cross-References" section linking to related traits. Key relationships:

- **Sleep deprivation** affects almost everything
- **Social class** influences education, profession, and politics
- **Profession** shapes vocabulary and problem-solving
- **Conspiracy involvement** interacts with ethics, technology view, and mental health

## Era Reminders

These traits are for 2026-2034 characters. Remember:

- Technology is 2020s-level (smartphones, social media, early AI)
- The "noise problem" is what brings them to Sleepless in Decibels
- Exhausted parents are the core demographic
- The conspiracy hasn't happened yet (or is happening)
- They don't know the world will end

For post-Silence characters (~2334 CE), use [post_silence_traits/](../post_silence_traits/).
